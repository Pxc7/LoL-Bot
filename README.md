# Installation

## Termux
```cmd
> pkg update && pkg upgrade
> pkg install git -y
> pkg install nodejs -y
> pkg install ffmpeg -y
```

## Cloning 
```cmd
> git clone https://github.com/Pxc7b/LoL-Bot
> cd LoL-Bot
```

## Install 
```cmd
> npm i
```

## Run the botv1
```cmd
> npm start
```
