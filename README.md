# Installation

## Termux
```cmd
> pkg update && pkg upgrade
> pkg install git -y
> pkg install nodejs -y
> pkg install ffmpeg -y
```

## Cloning 
```cmd
> git clone https://github.com/Pxc7/LoL-Bot
> cd LoL-Bot
```

## Install 
```cmd
> npm i
```

## Run the LoL-Bot
```cmd
> npm start
```
